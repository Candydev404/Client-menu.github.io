# 🚀 HQ Client Service Portal

A modern, interactive service menu designed to streamline client onboarding and project requests. Built with a premium "glassmorphism" aesthetic, this portal allows clients to view service details and instantly connect via direct WhatsApp routing or by opening a ticket in the private Discord server.

## 📋 Deployed Services Overview
The portal currently advertises the following core architectural capabilities:
* **Full-Stack Development:** Custom backend logic, secure user authentication, and responsive frontend interfaces.
* **Cybersecurity & Defense:** Web vulnerability scanning, server-side protection, and secure architecture.
* **Server Automation:** Intelligent scripts, Discord integrations, and deployment webhooks.
* **Custom GPT Bots:** *(Coming Soon)* AI-powered assistants tailored to specific business data.

## ✨ Key Features
* **Glassmorphism UI:** A sleek dark-mode interface with blurred backdrops and responsive grid layouts.
* **Dynamic Modals:** Vanilla JavaScript efficiently injects specific service data into a reusable modal component without requiring page reloads.
* **Direct Communication Routing:** Connects clients directly to specific WhatsApp links and Discord server invites.
* **Feature Locking:** Built-in logic to handle "Coming Soon" services with custom visual badges and disabled click events.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and FontAwesome icon integration.
* **CSS3:** Custom variables, flexbox/grid layouts, animations, and backdrop-filters.
* **JavaScript (ES6):** DOM manipulation, event listeners, and dynamic content rendering.

## 🚀 Deployment
This project is designed to be lightweight and can be hosted directly on **GitHub Pages**. 

1. Fork or clone this repository.
2. Update the `myWhatsAppLink` and `myDiscordInvite` variables in `script.js` with your own credentials.
3. Push to your `main` branch.
4. Navigate to your repository settings > Pages, and deploy from the main branch.
